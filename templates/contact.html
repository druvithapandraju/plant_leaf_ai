🌿 PlantCare AI

About This Project

This Plant Disease Detection system uses Deep Learning to identify plant diseases from leaf images. Farmers and gardeners can simply upload an image and get instant predictions.

The system is built using technologies like Flask, OpenCV, and TensorFlow/Keras. It helps in early detection of diseases, improving crop yield and reducing losses.

How It Works

1. Upload a leaf image
2. The model processes the image
3. It predicts the disease class
4. Results are shown instantly

Features

  • ✔ Upload plant leaf images
  • ✔ AI-based disease detection
  • ✔ Fast and accurate results
  • ✔ User-friendly interface
